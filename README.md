# Interactive Customer Churn Dashboard

## Project Overview
This project presents an interactive customer churn dashboard designed to analyze customer behavior, subscription cancellations, and retention patterns. The dashboard provides business insights through interactive visualizations, KPIs, and trend analysis to support data-driven decision-making.

---

## Objectives
- Analyze customer churn patterns
- Monitor customer retention metrics
- Identify factors influencing churn
- Visualize customer behavior trends
- Support strategic business decisions through interactive dashboards

---

## Dashboard Features

### KPI Metrics
The dashboard includes key performance indicators such as:
- Total Customers
- Churned Customers
- Active Customers
- Average Monthly Charges
- Average Customer Tenure

### Interactive Visualizations
The dashboard contains:
- Churn Distribution Analysis
- Contract Type vs Churn
- Internet Service vs Churn
- Monthly Charges Analysis
- Tenure Distribution
- Payment Method Analysis

### Filters and Slicers
Interactive filters allow users to explore data by:
- Gender
- Contract Type
- Internet Service
- Payment Method
- Churn Status

---

## Key Insights

### Customer Churn Trends
- Month-to-month contract customers showed the highest churn rate.
- Customers with higher monthly charges were more likely to churn.
- Customers with shorter tenure had higher cancellation probability.

### Service Insights
- Fiber optic internet users experienced higher churn rates.
- Electronic check payment users showed increased churn behavior.

### Retention Insights
- Long-term contract customers demonstrated stronger retention.
- Customers with lower monthly costs were more likely to remain active.

---

## Business Recommendations
- Encourage long-term contracts to improve retention.
- Offer targeted discounts for high-risk customers.
- Improve customer support for fiber optic users.
- Introduce loyalty programs for low-tenure customers.
- Optimize pricing strategies to reduce churn risk.
